# Presentacion interactiva para Dunia Institut Dental

Esta carpeta contiene una propuesta comercial en formato web interactivo para presentar un proyecto de agente de voz con IA para `Dunia Institut Dental`.

## Archivos

- `index.html`: estructura completa de la presentacion
- `styles.css`: identidad visual, layout y componentes
- `script.js`: interacciones, escenarios y simulador de ROI editable

## Como abrirla

La opcion mas simple es abrir `index.html` en el navegador.

Si prefieres servirla en local:

```bash
python3 -m http.server 4173
```

Despues abre:

```text
http://127.0.0.1:4173
```

## Que incluye

- Apertura con storytelling sobre la realidad de recepcion
- Flujo actual y coste oculto
- Flujo propuesto con agente de voz en Retell AI
- Escenarios A, B y C con comparativa
- Alcance claro de lo que resuelve, escala y no responde
- Ruta de reserva por WhatsApp con Doctoralia
- Ruta de cambio o cancelacion sin falsas automatizaciones
- Impacto para clinica y paciente
- Arquitectura simple y diferencial del proyecto
- Bloque final editable de inversion y ROI

## Personalizacion rapida

- Edita el contenido principal en `index.html`
- Ajusta el estilo visual en `styles.css`
- Cambia formulas o textos del simulador en `script.js`
